# Certification dispositif médical et déploiement production

## Étapes
1. **Soumission dossier** à un organisme notifié (ex. LNE, G-Med) : dossier technique EU MDR, gestion des risques, validation clinique.
2. **Audit** : revue du code, de l’architecture, des tests et de la sécurité.
3. **Obtention marquage CE** : délivrance du certificat pour Classe IIb.
4. **Déploiement production** : déploiement progressif (canary/blue-green) avec validation clinique à chaque étape.
5. **Surveillance post-commercialisation** : métriques (AUC, drift), vigilance (ANSM sous 15 j pour incidents graves), revues périodiques.

## Références
- EU MDR 2017/745
- EU AI Act (Annexe III §5)
- ISO 14971, ISO 13485
